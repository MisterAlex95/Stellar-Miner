# Add daily and weekly challenges

**Type:** feature
**Labels:** feature, quests
**Effort:** 3 points
**Status:** backlog

## Description
One daily quest (harder than normal, better reward) and one weekly (e.g. "Reach 100k total coins ever" or "Prestige 2 times"). Resets at midnight / week; uses existing quest UI with a "Daily" or "Weekly" badge.

## Acceptance Criteria
- [ ] Daily challenge is generated and resets at midnight (local or server).
- [ ] Weekly challenge is generated and resets weekly.
- [ ] Both use existing quest UI with clear Daily/Weekly badge and better rewards.

## Dependencies
None

- [ ] Integrate with quest panel and claim flow.
- [ ] Add badge and reward differentiation in UI.
