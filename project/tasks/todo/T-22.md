# Add achievement flavor text

**Type:** feature
**Labels:** feature, narrative
**Effort:** 1 points
**Status:** next-up

## Description
Add optional `flavor` (or `story`) to each entry in achievements.json. When unlocking, show that line in the unlock toast alongside the name. Example: Legend → "Your name is spoken in every port on the belt."

## Acceptance Criteria
- [ ] Achievements in data support optional flavor/story field.
- [ ] Unlock toast shows achievement name and flavor when present.

## Dependencies
None
