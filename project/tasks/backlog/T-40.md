# Theme-aware accents (game)

**Epic:** E-Game - Board engagement
**Type:** feature
**Effort:** 1 point

## Description
Planet/star colors or Empire visuals adapt slightly to light/dark theme. Cohesive feel.

## Acceptance Criteria
- [ ] At least one visual element (e.g. Empire panel, planet list, starfield) uses theme-dependent accent.
- [ ] No contrast or readability regression in either theme.
- [ ] Follows existing theme setting (light/dark).
