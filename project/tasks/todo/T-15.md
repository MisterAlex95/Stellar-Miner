# Add event flavor text

**Type:** feature
**Labels:** feature, narrative
**Effort:** 1 points
**Status:** next-up

## Description
Add optional `flavor` (or `storySnippet`) to each event in events.json. When an event triggers, show that line in the existing toast or a small popup. Optional setting to show/hide story toasts.

## Acceptance Criteria
- [ ] Events in data support optional flavor/storySnippet field.
- [ ] On event trigger, flavor is shown in toast or popup when enabled.
- [ ] Settings include toggle for story toasts.

## Dependencies
None

## Checklist
- [ ] Add flavor field to events schema and sample events.
- [ ] Pass flavor to event toast/popup in presentation.
- [ ] Add settings key and UI toggle for story toasts.
