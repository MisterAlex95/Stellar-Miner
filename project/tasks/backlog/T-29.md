# Add sound volume slider

**Type:** feature
**Labels:** feature, ui
**Effort:** 1 points
**Status:** inbox

## Description
If sound effects are enabled, add a volume slider (0–100%) in Settings, persisted with the rest. Allows quieter effects without turning them off. Requires AudioContext or HTMLAudioElement volume when sounds are added.

## Acceptance Criteria
- [ ] Settings include a volume slider when sound is enabled.
- [ ] Volume is persisted and applied to sound playback.

## Dependencies
T-10 (sound effects) recommended first.
