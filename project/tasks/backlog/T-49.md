# Daily login streak (game)

**Epic:** E-Game - Board engagement
**Type:** feature
**Effort:** 2 points
**Status:** inbox

## Description
Consecutive calendar days with at least one session; optional mulligan. Encourages return.

## Acceptance Criteria
- [ ] Game tracks consecutive calendar days with at least one load (login streak).
- [ ] Streak is displayed somewhere (Dashboard, Stats, or header) and persisted.
- [ ] Optional: one mulligan per week to not break streak when missing a day.
- [ ] Strings (en/fr).
