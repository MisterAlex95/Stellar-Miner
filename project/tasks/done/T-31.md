# Expedition composition (UI)

**Type:** feature
**Labels:** feature, ui
**Effort:** 2 points
**Status:** done

## Description
Let the player choose who goes when launching (e.g. "Send 4: 2 miners, 1 medic, 1 pilot"). Backend already supports launchExpedition(player, composition).

## Acceptance Criteria
- [ ] Before launching an expedition, player can set composition (e.g. 2 miners, 1 medic, 1 pilot).
- [ ] Selected composition is passed to launchExpedition and used by existing logic.
