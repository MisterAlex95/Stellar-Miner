**Epic:** E-Game - Board engagement
**Type:** feature

## Description
On one-page layout: swipe or 1–6 keys to switch tab. Faster navigation.

## Acceptance Criteria

- [ ] Optional: swipe left/right on touch to cycle tabs (if one-page supports it).
- [ ] Documented in Settings or help; does not conflict with existing shortcuts.
