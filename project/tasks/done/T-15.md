# Add event flavor text

**Type:** feature
**Labels:** feature, narrative
**Effort:** 1 points
**Status:** Done

## Description
Add optional `flavor` (or `storySnippet`) to each event in events.json. When an event triggers, show that line in the existing toast or a small popup. Optional setting to show/hide story toasts.

## Acceptance Criteria
- [x] Events in data support optional flavor/storySnippet field.
- [x] On event trigger, flavor is shown in toast or popup when enabled.
- [x] Settings include toggle for story toasts.

## Dependencies
None
