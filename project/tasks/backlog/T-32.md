# Add Specialists (one per type)

**Type:** feature
**Labels:** feature, crew
**Effort:** 3 points
**Status:** inbox

## Description
"Promote" one crew to Specialist (Miner / Scientist / Medic) for a coin cost. Only one specialist of each type. Miner: +5% production. Scientist: +10% research success. Medic: next expedition has -10% death chance (consumed when expedition launches). Unlock via research or milestone.

## Acceptance Criteria
- [ ] Player can promote one crew to each specialist type (Miner, Scientist, Medic) at a cost.
- [ ] Specialist bonuses apply as specified (production, research, expedition death).
- [ ] Medic effect is consumed on expedition launch.

## Dependencies
