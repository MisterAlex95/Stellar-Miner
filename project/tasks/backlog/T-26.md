# Remember Stats chart range

**Type:** feature
**Labels:** feature, ui
**Effort:** 1 points
**Status:** backlog

## Description
Persist the selected chart range ("Recent" vs "Long term") in localStorage (e.g. stellar-miner-stats-range). On load, restore selection so returning players don't have to click every time.

## Acceptance Criteria
- [ ] Chart range selection is saved to localStorage.
- [ ] On load, saved range is restored and applied to the chart.

## Dependencies
None

## Checklist
- [ ] On range change, write to localStorage.
- [ ] On Stats mount or load, read and apply saved range.
