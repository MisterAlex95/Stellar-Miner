# Add upgrade mastery or set bonuses

**Type:** feature
**Labels:** feature, upgrades
**Effort:** 2 points
**Status:** inbox

## Description
When the player owns at least one of every upgrade in a tier (e.g. all tier 1–3): grant a small bonus (e.g. +2% production). Encourages variety and revisiting early tiers.

## Acceptance Criteria
- [ ] Tier completion is detected (e.g. all T1–T3 owned).
- [ ] Set bonus is applied (e.g. +2% production) and reflected in production calc.
- [ ] UI indicates progress toward set bonus or when it is active.

## Dependencies
None

## Checklist
- [ ] Define tiers and set-bonus rules in catalog or config.
- [ ] Add check for "all in tier owned" and apply bonus in production formula.
- [ ] Show set bonus status in upgrade list or stats.
