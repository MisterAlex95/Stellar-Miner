# Show event countdown on badge

**Type:** feature
**Labels:** feature, ui
**Effort:** 1 points
**Status:** inbox

## Description
When an event is active, show remaining time on the event badge (e.g. "Meteor Storm ×2 — 18s"). Stats already has nextEventAt and event instances; pass endsAt to the badge or compute in eventBadge and display "X s left".

## Acceptance Criteria
- [ ] Event badge shows remaining time (e.g. "18s") while event is active.
- [ ] Uses existing event state (endsAt or computed from duration).

## Dependencies
None

## Checklist
- [ ] Compute or pass event endsAt to EventBadge component.
- [ ] Display countdown (e.g. "X s left") on the badge.
