# Add planet names and themes

**Type:** feature
**Labels:** feature, ui
**Effort:** 2 points
**Status:** inbox

## Description
Unlock a "theme" or rename for a planet at a cost (cosmetic only). Or random adjective + planet name (e.g. "Dust Haven — Prosperous"). Stored in save; no balance change.

## Acceptance Criteria
- [ ] Player can rename or theme a planet (cosmetic only).
- [ ] Custom name/theme is persisted in save.
- [ ] Display shows custom name/theme where planet is shown.

## Dependencies
None

## Checklist

- [ ] Add UI to set or unlock name/theme (e.g. in planet card).
- [ ] Use custom name/theme in all planet display locations.
