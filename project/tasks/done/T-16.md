# Add Codex or Archive

**Type:** feature
**Labels:** feature, narrative
**Effort:** 3 points
**Status:** Done

## Description
Unlockable lore entries (short texts) keyed to achievements, event types seen, planet types discovered, prestige level. Data: codex.json (id, title, body, unlockCondition). Application: codexUnlocks in state/save; on achievement/event/planet/prestige emit, call unlockCodexEntry(id). Presentation: Codex/Archive in Stats or Empire — list unlocked entries, grey out locked ones.

## Acceptance Criteria
- [x] codex.json exists with entries and unlock conditions.
- [x] Unlock logic runs on achievement, event seen, planet, prestige.
- [x] Codex/Archive view in Stats or Empire shows unlocked and locked entries.
