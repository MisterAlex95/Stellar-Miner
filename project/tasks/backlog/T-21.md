# Add Chronicle (journey log)

**Type:** feature
**Labels:** feature, narrative
**Effort:** 2 points
**Status:** inbox

## Description
Append-only log of key first-time moments (first upgrade, first planet, first prestige). Application: chronicle array in save — { date, eventId, sentence } per type; one narrative sentence per event. Presentation: "Chronicle" or "History" in Stats — scrollable list. Sentences can be generated from a small template map in code.

## Acceptance Criteria
- [ ] Chronicle entries are appended on first upgrade, first planet, first prestige, etc.
- [ ] Chronicle is persisted in save and shown in Stats (Chronicle/History section).
- [ ] Each entry has date, eventId, and a short sentence.

## Dependencies
None

## Checklist

- [ ] Define sentence templates for each event type.
- [ ] Add Chronicle/History view in Stats with scrollable list.
