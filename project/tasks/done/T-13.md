# Add combo decay indicator

**Type:** feature
**Labels:** feature, ui
**Effort:** 1 points
**Status:** Done

## Description
Show a small bar or countdown next to the combo that represents the 2.5 s window. So players see when the combo is about to drop and can click to maintain it.

- [ ] Combo display shows remaining time in the combo window (e.g. bar or "1.2 s").
- [ ] Uses COMBO_WINDOW_MS and existing combo logic.
- [ ] Indicator updates in real time (e.g. every 100 ms or on tick).
