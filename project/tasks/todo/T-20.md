# Add narrator or ship log toasts

**Type:** feature
**Labels:** feature, narrative
**Effort:** 2 points
**Status:** next-up

## Description
One-off lines from "the ship" or "mission control" on milestones (first 1M coins, first prestige, achievement:Legend). Data: narrator.json — list of { trigger, message }. Application: on eventBus/handlers, check if trigger already shown (narratorShown in state/save); if not, toast and mark shown.

## Acceptance Criteria
- [ ] narrator.json defines triggers and messages.
- [ ] On matching milestone, one-off toast is shown and trigger marked shown in save.
- [ ] No duplicate toasts for same trigger.

## Dependencies
None

- [ ] In handlers/eventBus, check triggers and show toast once per trigger.
- [ ] Add strings or use message from data.
