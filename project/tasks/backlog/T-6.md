# Add prestige milestones with small perks

**Type:** feature
**Labels:** feature, progression
**Effort:** 2 points
**Status:** backlog

## Description
At prestige 5, 10, 25, 50: unlock a small permanent perk (e.g. +1% offline cap, or "first quest of the day gives +10%"). Stored in progression; one perk per milestone keeps it simple.

## Acceptance Criteria
- [ ] Perks are defined for at least 4 prestige milestones (e.g. 5, 10, 25, 50).
- [ ] Perks are persisted in save/progression and applied in game logic.
- [ ] UI shows unlocked perks (e.g. in Stats or Prestige screen).

## Dependencies
None

## Checklist
- [ ] Define milestone levels and perk effects in data or config.
- [ ] Add progression field for unlocked milestone perks.
- [ ] Apply perks in relevant systems (offline, quests, etc.).
- [ ] Surface unlocked perks in UI.
