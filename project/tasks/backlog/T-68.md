# Export stats (game)

**Epic:** E-Game - Board engagement
**Type:** feature
**Effort:** 1 point
**Status:** inbox

## Description
Button to copy stats (or JSON) for sharing or backup. Power users.

## Acceptance Criteria
- [ ] In Stats tab or Settings, a button "Copy stats" (or "Export stats") copies a text or JSON summary of key stats to clipboard.
- [ ] Summary includes at least: total coins, prestige level, achievements count, main counters (clicks, quests, etc.).
- [ ] No sensitive data; format documented or human-readable.

## Dependencies
None
